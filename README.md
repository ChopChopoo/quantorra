# Quantorra

Synthetic-brand site for the Brand Visibility in Generative AI Engines
research project. **Group D** in the experimental design.

- Generated by `build_sites.py`. Do not edit the HTML by hand — edit
  `brands.py` and rerun the build.
- This README is for the researcher only; crawlers will see it because
  GitHub Pages serves the repo root as a directory listing if no
  `index.html` were present, but `index.html` exists so this README
  should not be indexed via the Pages site.
- Fingerprint phrase used for retrieval detection:
  `"the Quantorra dual-density heel"`. Grep T2 responses for this string.
